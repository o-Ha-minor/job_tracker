# Job Tracker

Job Trackerは、IT転職活動における求人応募の進捗を管理するためのRailsアプリです。

Railsアプリ作成の復習を兼ねて、実際の転職活動で利用できるよう、応募状況・面接日・志望度・懸念事項などを一元管理できるシンプルなCRUDアプリとして作成しました。

## 作成目的

複数企業へ応募する中で、選考状況や面接日、内定回答期限などの情報を整理しやすくするために作成しました。

また、Railsのscaffold、フォーム、i18n、CSSによる画面調整を復習することも目的としています。

## 主な機能

- 求人応募情報の作成・詳細表示・編集・削除
- 応募ステータスの管理
- 志望度の管理
- 応募日、一次面接日、二次面接日、最終面接日、内定回答期限の記録
- 懸念事項、質問事項、メモの記録
- 一覧画面での応募状況の確認
- i18nによるカラム名の日本語表示
- CSSによる一覧画面のスタイリング

## 使用技術

- Ruby
- Ruby on Rails
- SQLite3
- HTML
- CSS
- i18n

## 工夫した点

- scaffoldで作成したCRUD画面を、そのまま使わず求人応募管理向けに調整しました。
- `select` を使い、応募ステータスと志望度を選択式で入力できるようにしました。
- `human_attribute_name` と `ja.yml` を使い、一覧画面のカラム名を日本語で表示しました。
- 一覧画面では、求人比較に必要な項目に絞って表示し、見やすさを意識しました。
- Tailwind CSSなどのCSSフレームワークは使わず、Rails標準構成に近い形でCSSを記述しました。

## セットアップ

```bash
git clone https://github.com/o-Ha-minor/job_tracker.git
cd job_tracker
bundle install
bin/rails db:setup
bin/rails server
```

ブラウザで以下にアクセスします。

```text
http://localhost:3000/job_applications
```
